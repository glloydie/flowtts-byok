## üé§ Using FlowTTS BYOK

### Step 1: Create Your Tencent Cloud Account
To use FlowTTS BYOK, you need a Tencent Cloud account. Visit the Tencent Cloud website and follow their instructions to set up your account. 

### Step 2: Obtain Your API Key
Once your account is set up, navigate to the API key section on your dashboard. Create a new API key specifically for the FlowTTS BYOK application. This key is essential for using the text-to-speech service securely.

### Step 3: Input Your API Key
When you launch FlowTTS BYOK for the first time, you will be prompted to enter your API key. Paste the key from your Tencent Cloud account into the provided field.

### Step 4: Enter Your Text
After entering your API key, you will see a text box. Type or paste the text you want to convert into speech.

### Step 5: Choose Your Voice
FlowTTS BYOK allows you to select from multiple voice options. Click on the dropdown menu to choose your preferred voice for the output.

### Step 6: Generate Speech
Once you have typed in your text and selected a voice, click the "Generate" button. The application will process your request and produce an audio file of your text spoken in the selected voice.

### Step 7: Save or Play the Audio
After the audio is generated, you can either play it directly within the application or save it to your device. To save, click the "Save" button and choose a location on your computer.
